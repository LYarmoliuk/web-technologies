Інформація про використання структур даних

  • Map - зберігає продукти (ключ: ID продукту, значення: об'єкт продукту)
  • Set - зберігає унікальні категорії продуктів
  • WeakMap - відстежує історію змін продуктів (ключ: об'єкт продукту, значення: масив змін)
  • WeakSet - зберігає видалені продукти для потенційного відновлення
